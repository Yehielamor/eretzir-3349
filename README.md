# eretzir-3349

A real-time multiplayer Hebrew version of the classic 'Country, City' word game, where players compete to list Hebrew words for categories starting with a given letter within a time limit. The app features live game rooms, scoring, and a dictionary integration for validation, all designed for Hebrew speakers with an intuitive interface.

## Tech Stack
Vanilla HTML/CSS/JS + Supabase

## Setup
Run `docker-compose up` to start the infrastructure.
